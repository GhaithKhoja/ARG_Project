# ARG_Project

1. install python
2. install virtual environment
    python3 -m venv env 
3. activate venv
    source env/bin/activate
3. install require lib
    pip install -r requirement
    pip install -e .
4. run the script
    ./bin/apirun
5. navigate to the site
    e.g.(http://192.168.102.34:8000/)
